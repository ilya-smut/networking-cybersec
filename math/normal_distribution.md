# Normal distribution 
[![normaldist.png](https://i.postimg.cc/N0FcPbFn/normaldist.png)](https://postimg.cc/hXk6XL98)

## Propeties
σ - standart deviation
µ - mean (average)
N - size
X - subject

## Notation of normal distribution

N(0,1) - normal distribution

## Standard Z code

**Z = (X - µ) / σ**
